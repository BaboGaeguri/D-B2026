## 공통 평가 기준 (50점)

> 경영 교수님 / 데이터사이언스 교수님 공통 적용

| 평가항목 | 세부 기준 | 배점 |
|----------|-----------|------|
| **1. 문제 정의 및 주제 적절성** | 사회문제 또는 추천 과제를 명확히 정의했는가? | 15 |
| | 주제의 시의성과 중요성, 필요성이 설득력 있는가? | |
| **2. 해결 방안의 종합적 완성도** | 제안된 문제 해결 방향이나 전체적인 기획이 논리적인가? | 15 |
| | 해결 방안을 통한 전반적인 기대 효과가 명확하게 제시되었는가? | |
| | 기존 사례와 차별화된 접근이 있는가? | |
| **3. 창의성 및 독창성** | 새로운 아이디어나 시각이 반영되었는가? | 10 |
| **4. 문서 구성 및 표현력** | 보고서 구조가 체계적이고 시각 자료 활용이 효과적인가? | 10 |
| | 문장의 논리성과 가독성이 우수한가? | |

## 경영 전공 평가 기준 (50점)

| 평가항목 | 세부 기준 | 배점 |
|----------|-----------|------|
| **1. 비즈니스 타당성 및 실현 가능성** | 제안한 AI 도입/거버넌스 방안이 실제 기업 환경에서 실행 가능한가? | 15 |
| | 비용, 조직 구조, 리소스 측면에서 현실적인 검토가 이루어졌는가? | |
| **2. 경영 전략적 사고** | 경영학적 분석 도구를 활용하여 문제를 구조적으로 분석했는가? | 15 |
| | AI를 단순 기술이 아닌 비즈니스 자산으로 연결하는 전략적 논리가 있는가? | |
| | 이해관계자(고객, 경영진, 임직원 등) 관점이 전략에 반영되었는가? | |
| **3. 리스크 및 윤리적 고려** | AI 도입에 따른 경영 리스크(법적, 윤리적, 운영적)를 인식하고 있는가? | 10 |
| | 해당 리스크에 대한 구체적인 대응 방안이 제시되었는가? | |
| **4. 성과 및 임팩트 설계** | 제안 방안 실행 시 기대되는 비즈니스 성과가 구체적으로 제시되었는가? | 10 |
| | 성과의 측정 가능성(정량적 또는 정성적 지표)이 고려되었는가? | |
| **경영 전공 소계** | | **50** |
